# Topološka analiza podataka (TDA)

Glavni tekst se nalazi u `docs/main.pdf`.  
                      
Prateće jupyter sveske su:
- `src/homologija.ipynb` za prvu glavu
- `src/mapper.ipynb` za drugu glavu
- `src/perzistentna_homologija_pubmed_prosireno.ipynb` za primenu perzistentne homologije na PubMed skup (semantička i strukturna PH, poređenje po klasama, i proširenja: Persistence Image, pretraga, objedinjeni model)
                                
Za pokretanje koda je potrebno instalirati pakete iz `requirements.txt`.                
Najlakše preko `pip -r requirements.txt` unutar virtuelnog orkuženja.                     
Preporučena verzija python-a je `3.12`.
